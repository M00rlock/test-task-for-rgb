# RGB Test Project

Frontend:
- Next.js
- TypeScript
- Tailwind CSS
- shadcn/ui-style components

Backend:
- NestJS
- Prisma
- PostgreSQL

## Run locally

Frontend:
```bash
npm install
npm run dev
```

Backend:
```bash
cd backend
npm install
cp .env.example .env
npm run dev
```

PostgreSQL:
```bash
docker compose up -d postgres
```
